1D: Text Layer — Every Domain Has This

The foundation. Every single domain starts here. The concept description IS the 1D product. SEO longtail keywords drive traffic. API text responses power the tools. This is where Ollama + Claude generate value — transforming user input into structured text output.

concepts.json SEO keywords Ollama / Claude API Blog posts Meta descriptions JSON-LD schema

Example: toneswitch.com → input "make this email professional" → output rewritten text

2D: Visual Layer — Static Output

This is where the wordmap lives — and where it broke. The fix: generate visuals FROM the text layer programmatically. SVG wordmaps, OG social cards, architecture diagrams, brand color palettes. Every domain gets auto-generated 2D assets from its 1D data. Canvas API + SVG, no Photoshop.

Canvas API SVG generation OG image cards p5.js (static) Mermaid diagrams Color theming

Example: Auto-generate wordmap of fortune0 portfolio by category, color-coded, from concepts.json

3D: Interactive Layer — The MVPs

Working web tools. Input → processing → output. The 30 BUILD NOW domains get real interactivity. GitHub Pages serves the HTML, fortune0.com/api handles the heavy lifting (auth, AI, payments). Single-file HTML, zero dependencies, zero build step.

Motion: Audio/Video Layer — Time-Based

The richest layer. Whisper turns speech → text (MIT license, free commercial use). ffmpeg processes any media format (LGPL, dynamic link for commercial). Together they enable voice-first products, video analysis, podcast transcription, animated content. Client-side via whisper.cpp WASM or server-side via faster-whisper on the $7 Render box.

Teaching Taste — The Cringeproof Framework

Every fortune0 product teaches users to distinguish quality from noise. Cringeproof literally scores social skills. TweetAudit scores tweet quality. ToneSwitch teaches writing tone. The product IS the lesson. Across all four dimensions:

1D: Good taste in text

Clear copy, specific value props, honest descriptions. "AI pitch deck generator" not "Revolutionary synergistic platform."

1D: Bad taste in text

Buzzword soup, vague claims, keyword stuffing. The stuff that makes people cringe and bounce.

2D: Good taste in visuals

Clean wordmaps with real data. Category colors that mean something. Diagrams that explain, not decorate.

2D: Bad taste in visuals

The wordmap that got fucked up. Too many nodes, no hierarchy, colors that clash, impossible to read.

3D: Good taste in interaction

Single input, clear output. No signup wall. No loading spinners hiding nothing. Real value in 3 seconds.

3D: Bad taste in interaction

Fake processing delays, random scores, generic "insights." The template scaffolds before they become real products.

Motion: Good taste in A/V

Whisper transcription that actually helps. Short demo videos. Animations that show data, not just move.

Motion: Bad taste in A/V

Auto-play videos, stock footage, unnecessary transitions, voice-over that says nothing. Motion for motion's sake.
